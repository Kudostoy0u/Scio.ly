# Auth Directory

This directory contains authentication-related pages for the Scio.ly platform.

## Files

### `reset-password/page.tsx`
Password reset page component.

**Key Features:**
- Password reset form
- Token validation
- New password submission
- Error handling

**Important Notes:**
- Handles password reset flow
- Validates reset tokens
- Integrates with Supabase Auth
- Theme-aware design

## Important Notes

1. **Supabase Auth**: Uses Supabase authentication system
2. **Token Validation**: Validates password reset tokens
3. **Security**: Secure password reset process
4. **Theme Support**: Dark/light mode support
