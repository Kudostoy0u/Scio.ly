# Assign Directory

This directory contains the assignment system for the Scio.ly platform. Provides assignment functionality for team-based practice and assessment.

## Files

### `[id]/page.tsx`
Dynamic route for assignment pages by ID.

**Key Features:**
- Assignment display by ID
- Assignment completion interface
- Progress tracking
- Assignment submission

**Important Notes:**
- Dynamic route parameter: `id`
- Loads assignment data from API
- Supports assignment completion
- Integrates with team system

### `error/page.tsx`
Assignment error page.

**Key Features:**
- Error display
- Error recovery options
- User guidance
- Navigation back to assignments

**Important Notes:**
- Handles assignment-related errors
- Provides recovery options
- User-friendly error messages

## Important Notes

1. **Team Assignments**: Integrates with team system
2. **Dynamic Routes**: Supports assignment-specific pages
3. **Error Handling**: Comprehensive error handling
4. **Progress Tracking**: Tracks assignment completion
5. **Theme Support**: Dark/light mode support
