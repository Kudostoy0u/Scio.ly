# Crime Busters (2026)

Division: B  
Type: Lab  
Participants: 2  
Approx. Time: 50 minutes  
Allowed Materials: Safety goggles required; notes per rules

## Overview
Intro forensic chemistry: qualitative analysis of powders/liquids/polymers, chromatography, trace evidence, and reasoning.

## Core Topics
- Powder/liquid/polymer ID; pH and reaction observations  
- Chromatography basics; Rf  
- Fibers/hair; fingerprints; simple metals tests  
- Crime scene logic and reporting

## Skills
- Run spot tests safely and quickly  
- Keep organized observations and conclusions
