# Scio.ly Documentation Directory

## Overview

The `src/docs/` directory contains comprehensive documentation for Science Olympiad events, rules, and resources. This directory is organized by division (B and C) and provides detailed information about each Science Olympiad event.

## Directory Structure

### Division B Rules (`/division_b_rules/`)
- **Purpose**: Division B (middle school) Science Olympiad event rules and documentation
- **Content**: 36 markdown files covering all Division B events
- **Format**: Markdown documentation with event rules, parameters, and guidelines

### Division C Rules (`/division_c_rules/`)
- **Purpose**: Division C (high school) Science Olympiad event rules and documentation
- **Content**: 36 markdown files covering all Division C events
- **Format**: Markdown documentation with event rules, parameters, and guidelines

### Documentation Tracker (`todo.txt`)
- **Purpose**: Implementation status tracker for documentation
- **Content**: Status tracking for all Science Olympiad events
- **Format**: Plain text with status indicators
